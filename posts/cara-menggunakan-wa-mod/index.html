B. Backup Pesan

Sebelum beralih menggunakan Whatsapp Modifikasi, alangkah baiknya kita mencadangkan semua pesan ke cadangan lokal, sehingga bisa kita pulihkan di Whatsapp mod nantinya.

Langkah-langkah mencadangkan Pesan Whatsapp :

  • Buka Aplikasi Whatsapp
  • Tekan “Titik Tiga” di Pojok Kanan Atas
  • Pilih dan Buka Setelan
  • Masuk ke bagian “Chat”
  • Geser kebawah, dan tekan menu “Cadangan Chat”
  • Tekan Tombol “CADANGKAN” dan Tunggu sampai selesai ( 100% )
  • Setelah selesai, pastikan terdapat Informasi Waktu Terakhir Pencadangan.

C. Install Whatsapp Mod dan Memulihkan Pesan

Setelah semua pesan di Whatsapp original ( Sebelumnya ), telah di cadangkan ke Memori Lokal. Maka kita selanjutnya akan menginstall Whatsapp Mod untuk menggantikan Whatsapp Original sebelumnya.

SEBELUM MELAKUKAN LANGKAH INI.
PASTIKAN CHAT KAMU SUDAH BENAR-BENAR BERHASIL DI CADANGKAN !!!

Langkah-langkan Instalasi :

  • Uninstall Whatsapp Original ( Whatsapp yang sebelumnya dipakai )
  • Install Whatsapp Mod yang telah di Download
  • Setelah terinstall, buka Whatsapp dan Verifikasi Nomor Whatsapp kamu seperti biasanya.
  • Ketuk “PULIHKAN” ketika diminta untuk memulihkan chat dan media dari cadangan lokal.
  • Ketuk “CONTINUE” jika ingin memulihkan Pesan Whatsapp yang sebelumnya di cadangkan di Google Drive atau Ketuk “NOT NOW” jika ingin memulihkan Pesan Whatsapp yang sebelumnya di cadangkan di Memori Lokal

  • Selanjutnya, ketuk “RESTORE” dan Tunggu hingga Pesan dipulihkan 100%
  • Ketuk “INISIALISASI”
  • Selamat kamu sudah berhasil menginstall Whatsapp Mod

D. Penjelasan Menu Privasi Whatsapp Mod

Untuk memodifikasi atau menghidupkan konfigurasi privasi di Whatsapp Mod, kamu hanya perlu ketuk “Titik Tiga” di Pojok Kanan atas, lalu buka menu “Privasi”

  • Hide / Freeze Last Seen ( Bekukan Terakhir dilihat )
    Membekukan Status Terakhir dilihat
  • Anti Revoke ( Anti Hapus Pesan )
    Pesan terhapus, masih bisa dilihat
  • Blue Tick after Reply ( Sembunyikan Centang Biru hanya setelah balas )
    Perlihatkan Centang Biru ke kontak hanya ketika kamu telah membalas pesannya
  • Hide Status View ( Sembunyikan Melihat Status )
    Kontak tidak akan tahu, bahwa kamu telah melihat status dia
  • Hide Blue Tick ( Sembunyikan Centang Biru )
    Menyembunyikan Centang Biru dari kontak, walaupun kamu telah membaca pesannya
  • Hide Second Tick ( Sembunyikan Centang Dua )
    Menyembunyikan Centang Dua dari kontak, walaupun kamu telah menerima atau juga membaca pesannya
  • Always Online ( Selalu Online )
    Mengaktifkaan Status Terakhir dilihat menjadi selalu Online
  • Hide Forwarder Tag
    Menyembunyikan Simbol Pesan diteruskan
  • Anti-View Once
    Walaupun Gambar/Foto di setel hanya sekali lihat, namun kamu bisa selalu melihatnya
  • Anti-Delete Status
    Kamu akan selalu bisa melihat status kontak yang telah dihapus
  • Hide Blue Microphone ( Sembunyikan Microphone Biru )
    Sembunyikan Simbol Microphone Biru, walau kamu telah mendengarkan voice note dari kontak kamu
  • Hide Recording ( Sembunyikan Sedang Merekam )
    Sembunyikan notifikasi dan tidak memberi tahu kontak bahwa kamu Sedang Merekam
  • Hide Typing ( Sembunyikan Sedang Mengetik )
    Sembunyikan notifikasi dan tidak memberi tahu kontak bahwa kamu Sedang Mengetik

E. Penjelasan Menu Lanjutan/Advance Whatsapp Mod

Untuk memodifikasi atau menghidupkan konfigurasi lainnya di Whatsapp Mod, kamu hanya perlu ketuk “Titik Tiga” di Pojok Kanan atas, lalu buka gulir ke bawah dan pilih menu “Pengaturan”, lalu pilih “Pengaturan Lebih Lanjut”

  • Berbagi Media
    Kamu bisa memodifikasi konfigurasi tentang Media, seperti Kualitas Gambar, Video dan sebagainya.

  • Cadangkan & Pulihkan
    Fitur ini tidak terlalu berguna menurut saya pribadi, dan saya tidak pernah menggunakannya.

  • Bersihkan File Whatsapp Kamu bisa membersihkan Cache, Logs, dan Cadangan Obrolan lama. Sehingga bisa melegakan kapasitas memori internal HP kamu.

  • Pengaturan Lainnya Di dalam menu ini, kamu akan melihat beberapa konfigurasi seperti :
    a ) Sembunyikan Tombol Unduh :
    Menyembunyikan Tombol Unduh Media Status
    b ) Konfirmasi Pengiriman Sticker :
    Menampilkan POP-UP Konfirmasi, sebelum sticker di kirim
    c ) Ikon DND :
    Jika diaktifkan maka tombol Mode Pesawat atau Mode Offline akan di sembunyikan dari halaman depan

    Beberapa fitur atau menu mungkin tidak tersedia seperti diatas,  
karena mungkin Developer telah meperbaruinya atau menghilangkannya.
